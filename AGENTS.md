# Konsep Arsitektur Multi-Agent (Phenom AI)

Sistem ini dikembangkan menggunakan pendekatan multi-agent di mana setiap tugas dikelola oleh agen dengan spesialisasi tertentu.

## Daftar Agen

### 1. Agen UI/UX
- **Fokus**: Estetika, Tata Letak, Glassmorphism.
- **Tugas**: Merancang struktur komponen dan panduan styling (Tailwind 4).

### 2. Agen Frontend
- **Fokus**: Next.js 15, React 19, Lucide Icons.
- **Tugas**: Implementasi desain menjadi kode fungsional.

### 3. Agen Backend & AI
- **Fokus**: Prisma, OpenRouter, API Routes.
- **Tugas**: Mengelola database, logika scraping, dan prompt engineering.

### 4. Agen QA / Debugger
- **Fokus**: Validasi, Testing, Perbaikan Bug.
- **Tugas**: Memastikan kualitas kode dan memperbaiki error terminal/browser.

### 5. Agen DevOps
- **Fokus**: Vercel Deployment, Environment Variables.
- **Tugas**: Mengunggah aplikasi ke internet dan konfigurasi server.

---

**Catatan**: Skill Gemini CLI `multi-agent-architecture` telah diinstal untuk membantu koordinasi antar agen ini secara otomatis.
